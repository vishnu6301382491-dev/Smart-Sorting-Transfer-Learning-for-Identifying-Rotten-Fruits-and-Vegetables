🔍 Model Information

Technical details about our Smart Sorting AI model

🏗️ Architecture

Transfer Learning with Pre-trained CNN Model (VGG16/ResNet50)

🎯 Classes Supported

10 Classes: Apple, Banana, Tomato, Potato, Strawberry (Healthy & Rotten)

📐 Input Size

224×224 pixels (RGB images)

🛠️ Technical Specifications

Transfer Learning Base

Pre-trained ImageNet weights for feature extraction with fine-tuned top layers

Data Augmentation

Rotation, flipping, zooming, and brightness adjustments for robust training

Optimization

Adam optimizer with learning rate scheduling and early stopping

Loss Function

Categorical cross-entropy for multi-class classification

Training Dataset

5,000+ labeled images of fruits and vegetables in various conditions

Deployment

Flask API with real-time image processing and classification
